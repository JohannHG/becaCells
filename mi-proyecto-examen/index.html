Ejercicio 2

Escribir un programa que reciba dos strings y retorne el número de caracteres diferentes (comparando posición por posición). Nota: Puedes asumir que los strings siempre tienen la misma longitud. Sin embargo, si quieres agregarle más dificultad puedes pensar cómo solucionarlo si un string es más largo que el otro (la diferencia entre las longitudes agregaría al resultado).
CÓDIGO DE PRUEBA
  • console.log(distancia("hola", "hola")) // 0
  • console.log(distancia("sol", "tol")) // 1
  • console.log(distancia("carro", "correr")) // 3
